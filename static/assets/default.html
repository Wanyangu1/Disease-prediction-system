Why Disease Prediction?

We revolutionize healthcare by enabling early detection, personalized care, and cost savings. Empowering individuals with proactive insights, they pave the way for a healthier future and reshape the landscape of healthcare.

Learn More

Prediction Algorithm

The system uses a Decision Tree Classifier to predict diseases based on input symptoms.

Model Accuracy

The machine learning model accuracy stands at 92% with a vast range of predictable diseases

Dataset Used

The model has been trained using an upto date medical dataset extracted from Kaggle and contains columns and rows with symptoms and corresponding diseases

How the system works

The system analyzes input symptoms using machine learning algorithms, correlating patterns with known diseases. It leverages historical health data, identifies relevant features, and predicts the most probable disease. The patients can then book an appointment with an expert in the field of need for further consultation

Authentication

We use secure authentication methods to ensure security of sensitive health data, ensuring patient privacy, maintaining data integrity, and adhering to legal and ethical standards, fostering trust among users.

UI/UX

The system entails an intuitive website where users can view interact with different functions while enjoying efficient accesibility and usability

Backend

The frontend collects user-input symptoms, sending them to the backend via Flask, a web framework. The backend processes the data, predicts diseases using a ML model, and relays the feedback to the frontend for user presentation.
